# IBM Data Analyst Capstone Project – Developer Skills Analysis

## Overview

This project is part of the IBM Data Analyst Professional Certificate Capstone. The goal is to analyze current trends in the software development industry to identify the most in-demand technical skills.

As a data analyst working for a global IT consulting firm, the objective is to explore and extract insights from multiple data sources, including job postings, training platforms, and the Stack Overflow Developer Survey.

These insights will help identify emerging skill requirements in the tech industry and support data-driven decision-making for workforce planning and training strategies.

---

## Problem Statement

The technology industry evolves rapidly, and organizations need to continuously understand which skills are most in demand.

This project aims to answer key questions such as:

- Which programming languages are most commonly used and most desired?
- Which database technologies are trending?
- Which tools, platforms, and IDEs are most popular among developers?

---

## Data Sources

The main dataset used in this project is the **Stack Overflow Developer Survey (2024 subset)**.

Additional potential data sources include:

- Job postings data
- Online learning platforms
- Developer survey datasets

Official reference:
https://stackoverflow.blog/2024/08/06/2024-developer-survey/

---

## Project Workflow

The analysis will follow these steps:

1. Data Collection
2. Data Cleaning and Wrangling
3. Exploratory Data Analysis (EDA)
4. Statistical Analysis of trends
5. Data Visualization using IBM Cognos Analytics
6. Dashboard creation and storytelling presentation

---

## Dataset Description

The dataset contains responses from thousands of developers worldwide and includes information about:

- Programming languages
- Databases
- Web frameworks
- Platforms and tools
- Education and experience
- Employment and compensation
- AI tool usage

Each row represents a unique survey respondent.
